상태정보 확인

drwxr-xr-x 3 root root 4096 2018-04-09 16:14 work

  1. d : 디렉토리 여부 ( 디렉토리의 경우 : d , 아니면 -)

커맨드 목록

  • 파일 목록 확인
    • li
      • -l (list : 파일 표시)
      • -a (모든파일표시 (숨김파일표시 .(dot)으로 시작))
      • -S (크기별로 오름차순)
  • 현재 위치 확인
    • pwd
  • 디렉토리 생성
    • mkdir (make directory 의 축약형)
      • -p (경로 전체 디렉토리 생성)
  • 파일 생성
    • touch [파일명]
    • > [파일명]
  • 이동
    • cd (change directory)
  • 화면정리
    • clear
  • 삭제
    • rm (remove)
      • -r (디렉토리 삭제)
  • 상세 도움말
    • man [커맨드]
  • 간단 도움말
    • –help (옵션 값으로 사용)
